# Changelog

## v0.1.0

首个开源版本，聚焦中国联通云盘桌面端文件管理与本地传输体验。

### Added

- 文件管理：支持目录浏览、新建文件夹、重命名、移动和删除。
- 上传下载：支持文件上传、文件下载、默认下载目录和重名文件避让。
- 传输中心：上传/下载任务分 tab 展示，提供进度、速度、状态和批量操作。
- 多线程传输：支持下载/上传线程数、并发任务数、分片大小和重试次数配置。
- 断点续传：下载任务支持暂停、继续、取消、失败记录和续传状态展示。
- 安全凭据：网页登录态 Cookie 保存到系统安全凭据存储，不写入普通配置文件。
- 桌面界面：基于 PySide6 与 PySide6-Fluent-Widgets 提供 Fluent Design 体验。
- 工程化：加入 pytest、ruff、mypy，以及 GitHub Actions CI / release workflow。

### Notes

- 本项目不是中国联通或联通云盘官方客户端。
- 本项目不包含 OpenList 源码，OpenList 仅作为接口事实参考来源。
- Cookie、token、真实下载 URL、签名 URL 不应出现在日志、截图、文档或普通配置文件中。
